== Frequently Asked Questions ==

= How do I enable maintenance mode? =
Go to **Settings â†’ Easy Maintenance Mode**, and toggle the switch to "Enable Maintenance Mode."

= Can I customize the maintenance page? =
Yes! You can customize the message, background image, logo, and colors from the settings page.

= Will administrators still have access to the website? =
Yes, logged-in administrators can view the website normally while maintenance mode is active.

= Is this plugin SEO-friendly? =
Yes! The maintenance page sends a **503 Service Unavailable** status code to search engines, indicating that the website is temporarily down.
